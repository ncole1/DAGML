`self.DAG` is the list containing the entire DAG, with each line being a string element of the list. The process of importing the DAG involves a separation of the DAG into 3 components: `self.dagIndents`, which contains the number of indents at the beginning of each line of the DAG, `self.dagItem` contains the name of every operator, branch, and merge, where branch names begin with '<', and `self.dagFlows` which contains the flows of operators and branch heads, and sometimes also literals (a literal is an inbuilt argument of an operator). Literals appear to the left of a vertical bar in the flows for an operator. For operators and branch heads, the flows are not included in `self.dagItem` since they are part of `self.dagFlows`. All 3 of these components have exactly the same number of elements as the number of lines in the DAGML they represent, and correspond element by element. In addition, there is an integer `self.indentSize` which represents the number of space characters in a single indent. It applies to the entire DAG.
